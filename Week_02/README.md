### 树、二叉树

- 二叉树的遍历（前序\中序\后序遍历）

	- 前序遍历(pre_order)：根->左->右
	- 中序遍历(in_order)：左->根->右
	- 后序遍历(post_order)：左->右->根
	- 层次遍历：只需按层次遍历即可
